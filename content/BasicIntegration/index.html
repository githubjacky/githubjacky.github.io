Basic Integration

Introduction

This project provide and interface for solving integration problems using guass quadrature and monte carlo integratioon no matter the problem is univariable or multivarialbe. Furthermore, based on interest, I also implement the function for importance samplingin quasi way through halton sequences.

For Gauss Quadrature method I implement:

  • 1. Gauss Hermite
  • 2. Gauss Laguerre
  • 3. Gauss Legendre

As for monte carlo method I implement:

  • 1. naive monte carlo
  • 2. quasi monte carlo
  • 3. importance samplng

note: both the quasi monte carlo and importance sampling using halton sequencs to implment
